Changelog

New updates and improvements to .env

v0.1.0 Latest
December 29, 2025

Added

  • Initial release
  • LLM sandbox with model management
  • Agent orchestration interface
  • Native macOS performance optimizations
  • Support for Claude, GPT-4, Gemini, and Grok
  • Context Sharing - Connect nodes to share conversation context between chats, enabling linked nodes to reference parent messages for complex multi-step reasoning
  • Tool Use/Function Calling - Built-in file system tools (read, write, list, delete) and shell command execution for Claude and OpenAI, securely executed through Electron's IPC bridge
  • Session Management - Save and load canvas states with all nodes and connections, auto-save with 1-second debouncing, multiple sessions via sidebar
